# LightPilot 灯光布置辅助系统

一个静态前端原型，用于根据现场图片、场地尺寸、活动类型和灯具数量生成灯光布置建议。当前版本不依赖后端，直接打开 `index.html` 或启动本地静态服务即可使用。

## 当前能力

- 上传现场照片或平面图作为平面参考底图
- 输入场地宽度、进深、层高、活动类型和环境光条件
- 配置面光、聚光、摇头、染色、轮廓、灯带数量
- 自动生成平面布点、简化 3D 预览、覆盖评分和风险提示
- 输出灯具清单、布置建议和 JSON 方案

## 本地运行

直接打开：

```bash
open index.html
```

或启动本地服务：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```
